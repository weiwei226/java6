# java6
基于jsp+mysql的网络教学平台源码代码（设计文稿+答辩PPT+开题报告）
>  **运行环境:** jdk 1.8  
>  **IDE环境：** Eclipse,Myeclipse,IDEA都可以  
>  **硬件环境：** windows 7/8/10 1G内存以上  
>  **用了技术框架：** jsp+mysql  
>  **可以实现：** 管理员对全局的管理，比如用户，老师，题目，留言板，发布资料等。老师可以实现在线发布题目，下载学生作业等。学生可以浏览资料，公告，发表留言等。总之完整的实现了一个网络教学平台的功能。
